# 화면별 데이터 안내

각 대시보드 화면이 어떤 데이터를 보여주는지 정리합니다.
개발자가 아니어도, 화면의 숫자가 어디서 오는지 이해할 수 있도록 작성했습니다.

---

## 이상 신호 배너

모든 대시보드 상단에 표시됩니다. 시스템이 자동으로 감지한 이상 패턴이 있으면 빨간색~파란색 배너로 알려줍니다.

표시하는 것:
- 출고량이 갑자기 늘거나 줄었을 때
- 반품이 갑자기 많아졌을 때
- 특정 품목 재고가 비정상적으로 변했을 때
- 특정 채널 매출이 급락했을 때

클릭하면 해당 상세 화면으로 바로 이동합니다.

---

## 공급망(SCM) 화면

### 재고 현황
지금 팔 수 있는 재고가 창고별, 품목별로 얼마나 있는지 보여줍니다.

### 재고회전율
재고가 얼마나 빨리 팔리고 보충되는지 보여줍니다. 회전율이 낮으면 재고가 오래 쌓여있다는 뜻입니다.

### 품절 위험
현재 판매 속도 기준으로 재고가 며칠 뒤에 떨어지는지 계산합니다. 기준 일수보다 빨리 떨어질 품목을 빨간색으로 표시합니다.

### 미입고 발주
발주는 했지만 아직 입고되지 않은 수량과 예상 도착일을 보여줍니다.

### 리드타임
공급사에 발주한 뒤 물건이 도착하기까지 실제로 걸린 기간을 분석합니다.

### 출고/반품
날짜별 출고 수량과 반품 수량 추이를 보여줍니다. 급격한 변동이 있으면 이상 신호 배너에도 표시됩니다.

### 병목 현황
공급망 전체에서 문제가 되는 지점(납기 지연, 재고 부족 등)을 자동으로 찾아서 보여줍니다.

### 예측
과거 데이터를 기반으로 향후 수요를 예측하고, 보충 발주 권장량을 제시합니다.

---

## 손익(P&L) 화면

### 매출
기간별 매출을 채널, 품목 기준으로 보여줍니다.

### 매출원가(COGS)
판매한 물건의 원가를 계산합니다. 출고 수량과 반품을 반영한 순수 원가입니다.

### 공헌이익
매출에서 매출원가와 변동비를 빼고 남는 이익입니다. 채널별로 실제 돈을 벌고 있는지 보여줍니다.

### 영업이익
공헌이익에서 고정비까지 빼고 남는 최종 이익입니다.

### 수익성 순위
품목별, 채널별 수익성을 순위로 비교합니다.

---

## 필터가 작동하는 방식

- 기간 선택: 월 단위(2025-03) 또는 날짜 범위(3/1~3/31)로 필터합니다.
- 창고/품목/채널/공급사: 선택하면 해당 조건에 맞는 데이터만 표시됩니다.
- SCM 화면은 날짜 기준, P&L 화면은 월(period) 기준으로 데이터를 읽습니다.

## 알아둘 점

- 화면에는 정리된 분석 데이터만 표시됩니다.
- 업로드한 원본 데이터나 내부 처리 데이터는 화면에 직접 나오지 않습니다.
