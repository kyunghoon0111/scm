# 확장 로드맵

이 문서는 Phase 1 이후의 전체 확장 방향을 정리한 것입니다.
각 Phase별 세부 세션 가이드는 별도 문서에 있습니다.

## Phase 1.5 — 안정화 (5세션)

현업이 실제로 쓸 수 있는 수준으로 만든다.

1. 엑셀 다운로드
2. 필터 드롭다운
3. 페이지네이션
4. CI/CD + 에러 트래킹
5. 기본 테스트

가이드: `docs/phase1.5-session-guide.md`

## Phase 2 — 데이터 심화 (5~6세션)

SCM 의사결정에 필요한 핵심 지표를 추가한다.

1. 재고회전율
2. ABC 분류
3. 공급사 스코어카드
4. 채널별 수익성
5. 파이프라인 알림

가이드: `docs/phase2-session-guide.md`

## Phase 3 — 운영 관리 UI (4세션)

관리자가 데이터 이력을 추적하고 문제를 직접 처리할 수 있게 한다.

1. 업로드 이력 조회
2. 재처리 (단계별 파이프라인 실행)
3. DQ 대시보드
4. 감사 로그

가이드: `docs/phase3-session-guide.md`

## Phase 4 — 분석 고도화 (4~5세션)

"숫자를 보여주는 대시보드"에서 "의사결정을 돕는 분석 도구"로 진화한다.

1. Drill-down 네비게이션
2. 기간 대비 비교 분석
3. 커스텀 리포트 빌더
4. 원인 분석 강화

가이드: `docs/phase4-session-guide.md`

## Phase 5 — 자동화/알림 (4~5세션)

사람이 대시보드를 보지 않아도, 시스템이 이상을 감지하고 알려준다.

1. 이상치 탐지
2. 알림 시스템
3. 추천 액션
4. 스케줄 배치 자동화

가이드: `docs/phase5-session-guide.md`

## 전체 예상

- 총 세션: 22~25회
- Phase 1.5~2 완료 시: 실무 사용 가능한 SCM 대시보드
- Phase 3 완료 시: 운영 자립 가능
- Phase 4~5 완료 시: 분석 + 자동화 플랫폼
